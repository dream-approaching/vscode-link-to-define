### 更新日志

#### [v0.1.10] - 2020-11-10

- 支持小程序项目中 `path` 的跳转，包含
  - `WxService.navigateTo(path)`
  - app.json 中的 path
  - ext.json 中的 path
  - wxml 中的图片 path

#### [v0.1.9] - 2020-11-09

- 支持小程序项目中 `wxml` 和 `json` 文件中组件的点击跳转

#### [v0.1.7] - 2020-11-02

- webapp 项目和小程序项目可跳转`ajax api`
